# 对白追加翻译

为角色回复中的对白追加目标语种翻译，显示格式如下：

`"……我知道了。"「……わかった。」`

## 特点

- 在扩展页内配置
- 直接复用已有 Connection Profile
- 在角色消息渲染后自动处理
- 不修改原始 `message.mes`
- 将结果缓存到 `message.extra.jp_dialogue_append`
- 支持自定义目标翻译语种
- 支持名称翻译映射
- 支持自定义对白匹配正则
- 支持独立编辑副模型提示词
- 支持分批翻译，减少长对白被截断的问题

## 当前实现

- 默认支持 `「……」` / `『……』` / `"..."` 三种对白格式
- 额外支持用户自定义正则
- 使用所选 Connection Profile 调用副模型
- 将追加后的文本写入 `message.extra.display_text`
- 当对白较多时，可按“每批最大对白条数 / 每批最大字符数”自动拆分后逐批翻译
- 目标语种可填写语言代码或自然语言名称，例如：`ja`、`日语`、`en`、`英语`、`ko`、`韩语`

## 建议设置

如果你遇到“只翻译一半”的问题：

1. 先把 **最大输出 Token** 调高，例如 `4000`、`6000` 或更高。
2. 保持 **启用分批翻译**。
3. 适当降低：
   - 每批最大对白条数
   - 每批最大字符数

## 注意

当前版本为了稳定显示，使用 `display_text` 作为渲染入口，因此如果和会同样覆盖 `display_text` 的其他扩展同时启用，可能出现显示层竞争。
